How to connect and run on Webots

on Webots, open the world by navigating to this directory's webits_sim folder and open "football_field"

On your prefered IDE, Create 2 windows:

    window 1:
        go to ugv_sim_code
        run python3 sim_ground_station.py

    window 2:
        go to uav_sim_code
        run the virtual environment: .venv\Scripts\activate
        run the test: .\.venv\Scripts\python.exe .\[TEST SCRIPT NAME]

once you get window 2 running, it should be showing on webots
